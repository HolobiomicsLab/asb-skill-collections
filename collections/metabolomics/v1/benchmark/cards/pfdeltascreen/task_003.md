## Workflow Description
1. Parse input feature list containing m/z values. 2. Calculate Kendrick mass for each feature using the formula: Kendrick mass = (nominal mass / exact mass) × exact mass, where exact mass is converted to the Kendrick scale (e.g., CH₂ = 14.0157 Da). 3. Compute Kendrick mass defect (KMD) as the difference between nominal Kendrick mass and exact Kendrick mass. 4. Group or flag features with equivalent or near-equivalent KMD values (within user-specified tolerance) to identify homologous series members. 5. Output annotated feature table with KMD values and series group assignments.

## Domain Knowledge
- Kendrick mass defect is calculated by converting exact m/z to the Kendrick scale (typically using CH₂ = 14.0157 Da as the reference unit) and computing the difference between the nominal and exact Kendrick mass; features with identical KMD values represent homologous series differing by integer multiples of CH₂ units.
- PFAS compounds often form homologous series with repeating CH₂ units (chain extensions), making KMD analysis a key signature for identifying potential PFAS candidates in non-target screening.
- The choice of reference unit for Kendrick scaling (e.g., CH₂ vs. CF₂) affects the KMD pattern; for PFAS screening, CH₂-based Kendrick mass is commonly used to detect hydrocarbon-chain-extension homologs.
- KMD grouping tolerance (the allowed deviation from exact KMD equality when clustering features into series) typically ranges from 0.001 to 0.005 Da and depends on instrument mass accuracy and resolving power.
