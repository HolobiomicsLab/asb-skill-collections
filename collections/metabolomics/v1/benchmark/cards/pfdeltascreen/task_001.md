## Domain Knowledge
- Centroiding is a preprocessing step that converts profile-mode MS data into discrete m/z–intensity pairs; pyOpenMS expects centroided input for FeatureFinderCentroided.
- Feature detection identifies groups of consecutive MS scans sharing similar m/z values and intensity patterns, forming 2D peaks in the m/z–retention time plane.
- The output feature list must include at least three attributes per feature: monoisotopic m/z, elution time (retention time), and integrated or apex intensity.
- mzML is a standardized, vendor-neutral XML format for mass spectrometry data that encodes spectral metadata, scan order, and peak data in a machine-readable structure.
