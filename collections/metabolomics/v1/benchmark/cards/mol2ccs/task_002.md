## Domain Knowledge
- Feature ablation iteratively removes or masks input features and measures prediction change; gradient-based saliency uses backpropagation to compute input-feature sensitivity.
- CCS (collision cross section) is an instrumental property sensitive to molecular size, shape, and charge state; structural features include atomic number, hybridization, formal charge, bond type, and topology.
- GNN node features typically encode atom properties; edge features encode bond properties and connectivity; attribution must distinguish local atomic contributions from global graph structure effects.
- Aggregating attribution scores across a diverse test set requires accounting for molecular size variation (larger molecules have more features) and using relative or normalized importance metrics.
