## Domain Knowledge
- SMILES strings represent molecular structures as linear notation; canonicalization ensures consistent representation of the same molecule across multiple input formats.
- Atom features typically include atomic number, formal charge, hybridization state, and aromaticity; bond features include bond type (single/double/triple/aromatic), stereochemistry, and conjugation status.
- RDKit is the standard open-source library for SMILES parsing, molecular graph construction, and feature extraction in cheminformatics.
- Graph neural networks for molecular property prediction consume node feature matrices (atoms) and edge feature matrices (bonds) as input, requiring proper tensor serialization.
- Feature tensor formats must match the downstream GNN architecture's expected input shape and normalization conventions.
