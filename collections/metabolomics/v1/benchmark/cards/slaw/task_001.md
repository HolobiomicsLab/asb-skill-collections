## Workflow Description
1. Load aligned feature table with missing intensity values (NA or zero entries) from prior alignment step. 2. Apply data recursion algorithm to identify features present in some samples but absent (below detection) in others. 3. For each missing feature intensity, search recursively through related samples (by retention time and m/z proximity) to locate intensity values in alternate sample batches or replicates. 4. Fill missing intensities using the recursively retrieved values, maintaining sample-feature matrix structure. 5. Output completed feature table with filled intensities to CSV or mzTab format.
