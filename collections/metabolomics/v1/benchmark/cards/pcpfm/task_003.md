## Domain Knowledge
- Missing values in metabolomics data represent features not detected in certain samples and must be imputed rather than left as zero to avoid bias in downstream statistical testing.
- The interpolation_ratio parameter is a multiplier applied to each feature's minimum non-zero value, allowing user control over imputation stringency; lower ratios (e.g., 0.5) produce more conservative estimates.
- Moniker-based table naming enables tracking of multiple versions of the same feature table through the pipeline without overwriting intermediate results.
- In mass spectrometry metabolomics, feature intensity errors are often multiplicative, making minimum-value-based imputation more appropriate than constant imputation across features.
