## Domain Knowledge
- Ion status categories in mpactr include mispicked (isotopic pattern splitting errors), group (solvent/media blank contamination), replicability (non-reproducible across technical replicates), insource (fragment ions created during ionization), and passed (ions meeting all filter criteria).
- The qc_summary() function in mpactr returns a data.table where each row represents an input ion and status column indicates which filter(s) the ion failed or if it passed all filters.
- Treemaps display hierarchical data where rectangle area is proportional to ion count and color intensity (Greens palette) represents relative abundance or importance within each status category.
- Reference semantics in R6 objects used by mpactr enable in-place data updates without copying, making memory-efficient chaining of filters and downstream analysis operations.
- Ion filtering workflow typically proceeds in order: mispicked ions → blank contamination → replicability → insource fragments, allowing high-quality ions to be identified before downstream analysis.
