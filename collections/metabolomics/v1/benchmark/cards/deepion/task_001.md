## Connected Finding

## Domain Knowledge
- COL mode is designed for co-localized ions where spatial proximity relationships are preserved through augmentation, while ISO mode targets isotope patterns where intensity distributions vary systematically.
- Poisson noise augmentation simulates photon-counting statistics inherent to mass spectrometry detector responses.
- Intensity-dependent missing value in ISO mode must selectively remove high or low intensity pixels to simulate isotope abundance patterns, distinguishing it from uniformly random missing values in COL mode.
- Color jitter adjusts pixel intensities across the ion image globally; filtering operations (e.g., Gaussian, median) modify local neighborhood statistics.
