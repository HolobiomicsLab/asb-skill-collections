## Domain Knowledge
- ResNet18 is a residual convolutional neural network with 18 layers that enables training of deeper architectures by using skip connections to mitigate gradient vanishing.
- Shared-weight encoder pairs enforce identical parameter sets across both branches, ensuring that identical augmented images produce identical representations and different augmentations of the same image produce nearby representations in the learned space.
- The 512-dimensional output is a learned bottleneck representation that compresses spatial ion image information into a fixed-size feature vector suitable for contrastive loss computation.
- Parameter sharing reduces model complexity and training overhead while enforcing the symmetry assumption required for contrastive learning frameworks that compare augmentation pairs.
