## Workflow Description
1. Load metabolite intensity matrix (rows=metabolites, columns=samples) and pathway database (metabolite set definitions). 2. For each pathway, apply PLAGE (Pathway Level Analysis of Gene Expression) decomposition via singular value decomposition (SVD) on the subset of metabolites assigned to that pathway. 3. Extract the first principal component (singular vector) from each pathway's SVD as the pathway activity score. 4. Scale and normalize pathway scores across all pathways. 5. Rank pathways by activity level magnitude and compile results into a table with pathway identifiers, activity scores, and statistical significance metrics.

## Domain Knowledge
- PLAGE (Pathway Level Analysis of Gene Expression) uses singular value decomposition on pathway-specific metabolite subsets to extract the dominant activity pattern as the first principal component.
- The first singular vector from SVD of a pathway's metabolite intensity subset serves as the pathway activity score, representing coordinated metabolite behavior within that pathway.
- PALS decomposition is robust to noise and missing peaks in metabolomics data, a critical advantage over alternative methods like ORA and GSEA when peak detection is imperfect.
- The method is generalizable to any group of metabolite sets (not restricted to canonical pathways), enabling flexible user-defined metabolite groupings.
- Pathway activity scores must be scaled and normalized appropriately to allow cross-pathway ranking and comparison on a common scale.
