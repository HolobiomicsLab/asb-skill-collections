## Domain Knowledge
- PLAGE (Pathway Level Analysis of Gene Expression) is a singular value decomposition–based method that decomposes activity levels across feature sets and is more robust to noise and missing peaks than ORA and GSEA alternatives.
- The PALS decomposition approach is agnostic to set type and can analyze any group of metabolite sets, not limited to pathways, enabling custom user-defined metabolite groupings.
- Metabolite set files must be validated for consistent identifier format and proper set-membership structure before dispatch through the PLAGE pipeline to ensure compatible scoring.
- PALS Viewer currently ships with three predefined set types (pathways, Molecular Families from GNPS, and Mass2Motifs from MS2LDA); user-uploaded sets represent a functional extension beyond these.
