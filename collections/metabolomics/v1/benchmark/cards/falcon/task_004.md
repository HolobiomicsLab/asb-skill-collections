## Domain Knowledge
- Density-based clustering (e.g., DBSCAN) groups spectra by local density rather than global distance, making it suitable for spectra of varying cluster sizes and irregular shapes.
- A sparse pairwise distance matrix contains only non-zero distances for nearest neighbor pairs, reducing memory and computation compared to a full distance matrix.
- Cluster assignment output must preserve spectrum identifiers to enable downstream interpretation and cross-referencing with the original spectral data.
- MS/MS spectra similarity is typically measured using cosine distance or spectral angle after binning and feature hashing normalization.
