- MSMetaEnhancer adds metadata fields including SMILES, InChI, and CAS number through asynchronous queries to external chemical databases.
- The annotation pipeline supports both web-based converters (CIR, CTS, PubChem, IDSM, BridgeDb) that call external APIs and compute-based converters (RDKit) for local chemical structure transformations.
- The Logger component tracks per-attribute conversion success/failure events; fill-rate is the proportion of spectra records with a non-null value for a given metadata field.
- The .msp (NIST MS Search) format is a text-based spectral library format where each entry contains a spectrum and associated metadata fields (e.g., Name, PrecursorMZ, Compound_Name, SMILES).
- Annotation coverage improvement is measured as the change in fill rate from baseline (input file) to final (post-annotation), quantifying the enrichment value delivered by the converter pipeline.
