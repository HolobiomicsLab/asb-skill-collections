## Research Question
What metadata fields can MSMetaEnhancer add to mass spectra records by querying external web services?

## Connected Finding
MSMetaEnhancer enriches .msp files by adding SMILES, InChI, and CAS number metadata retrieved from five external services: CIR, CTS, PubChem, IDSM, and BridgeDb.

## Workflow Description
1. Load a sample .msp spectra file into the MSMetaEnhancer Application class. 2. Configure conversion jobs specifying source attributes (e.g., compound name), target attributes (e.g., SMILES, InChI, CAS), and converter services (CIR, CTS, PubChem, IDSM, BridgeDb). 3. Invoke the asynchronous `annotate_spectra` method with the defined jobs to dispatch metadata enrichment requests to web services. 4. For each job, the WebConverter base class calls `query_the_service` to make HTTP requests to the target API endpoint. 5. Parse API responses using service-specific `parse_response` methods to extract SMILES, InChI, CAS and other metadata fields. 6. Aggregate enriched metadata back into the spectra records. 7. Save the annotated spectra to an output .msp file.

## Domain Knowledge
- MSMetaEnhancer's Application class provides an asynchronous `annotate_spectra` method that accepts conversion jobs as tuples of (source_attribute, target_attribute, converter_name) to control metadata enrichment workflow.
- Web converters (CIR, CTS, PubChem, IDSM, BridgeDb) inherit from WebConverter base class and implement `query_the_service` for HTTP requests and `parse_response` for extracting structured metadata from API responses.
- .msp format is a text-based mass spectrum metadata file format where compound annotations (SMILES, InChI, CAS) are stored as key-value pairs that MSMetaEnhancer appends during enrichment.
- Asynchronous execution in MSMetaEnhancer optimizes fetching speed by allowing concurrent requests to multiple web services rather than sequential blocking calls.
- Conversion jobs must be defined with valid source attributes present in input .msp records and target attributes supported by the selected converter services to ensure successful metadata population.
