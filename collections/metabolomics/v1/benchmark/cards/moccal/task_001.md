## Domain Knowledge
- TWIM platforms record arrival time (time ion reaches detector) rather than drift time (time spent in TWIM cell), requiring explicit conversion for CCS calculation.
- CCS calibration requires a set of reference compounds with known collision cross section values to establish the arrival-time-to-CCS mapping function.
- Class-specific calibration is necessary because different biomolecular classes (proteins, lipids, metabolites) may have distinct arrival-time-to-CCS relationships.
- Calibration templates must contain arrival-time measurements for the same reference compounds run under identical instrument conditions as the experimental data.
