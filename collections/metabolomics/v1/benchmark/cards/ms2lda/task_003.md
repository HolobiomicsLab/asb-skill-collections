## Domain Knowledge
- Spec2Vec computes cosine similarity between spectral embeddings learned from deep neural networks, enabling annotation transfer from spectral libraries to unannotated motifs.
- Mass2Motif Annotation Guidance (MAG) integrates Spec2Vec similarity scoring to suggest structural labels for discovered motifs by matching against known spectral references.
- The Zenodo deposit (zenodo.org/records/15688609) contains the pre-trained Spec2Vec model optimized for positive-mode tandem mass spectrometry data; negative-mode spectra require separate models.
- Similarity score thresholds determine annotation confidence; higher scores (closer to 1.0) indicate stronger evidence for a proposed structural label.

## Methodology Summary
1. Load pre-trained Spec2Vec model and spectral embeddings from Zenodo deposit.
2. Parse trained motifset JSON file to extract all Mass2Motif records and their fragment/loss compositions.
3. Compute cosine similarity between each motif's embedding and reference spectral embeddings using Spec2Vec.
4. Rank candidate annotations by similarity score and extract top-N matches with associated structural labels.
5. Validation: Output JSON contains all motif identifiers with non-null similarity scores and at least one annotation label per motif; similarity scores fall within [0, 1] range and are ranked in descending order.
