## Domain Knowledge
- LC-MS feature tables contain m/z (mass-to-charge ratio) and retention time (RT) as primary identifiers for metabolite features across datasets.
- Feature matching relies on tolerance windows for both m/z and RT dimensions; typical metabolomics workflows use ppm-based m/z tolerance (5–10 ppm) and RT tolerance in seconds.
- M2S is a specialized Matlab implementation designed to solve the feature correspondence problem in untargeted metabolomics, where the same metabolite may appear at slightly different m/z and RT values across runs or instruments.
