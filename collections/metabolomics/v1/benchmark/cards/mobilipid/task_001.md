## Domain Knowledge
- The DTCCSN2 library contains collision cross section (CCS) values measured under nitrogen gas (N2) for uniformly labeled U13C lipids, serving as an internal calibration standard.
- CCS values in ion mobility spectrometry are sensitive to lipid structure, charge state, and measurement conditions; libraries must specify the pressure and gas type under which measurements were conducted.
- Lipid class coverage refers to the diversity of structural classes (e.g., PC, PE, PS, TG) represented in the reference library; effective CCS bias correction requires at least a few representatives per class.
- The U13C uniform labeling allows internal standardization by coelution with native lipids, eliminating the need for external calibration measurements beyond vendor-specific system tuning.
