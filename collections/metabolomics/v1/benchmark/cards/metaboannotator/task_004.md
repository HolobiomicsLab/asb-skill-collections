## Domain Knowledge
- MSP (NIST MS Search) format stores spectral metadata and peak lists; conversion to CSV requires extraction of compound name, precursor m/z, retention time, and peak intensity tuples with proper ionization mode notation.
- Peak-picking noise threshold (0.005) filters spectral peaks below relative intensity 0.5%, while marker peaks threshold (0.1) identifies diagnostic fragment ions representing ≥10% base peak intensity.
- Positive ([M+H]+) and negative ([M-H]−) ionization mode designations must be retained in library entry identifiers to enable correct spectral matching during annotation workflows.
- MassBank is a curated public MS/MS spectral database with standardized metadata; the bundled example subset validates import integrity without requiring external network access.
