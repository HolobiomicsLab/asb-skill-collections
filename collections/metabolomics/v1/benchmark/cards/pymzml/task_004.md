- Indexed gzip files are compressed mzML files with embedded random-access indices that allow seeking without full decompression, whereas standard gzip requires sequential reading.

- SQLiteDatabase wrapper must instantiate a sqlite3 connection and implement cursor queries to retrieve spectrum XML elements by ID, then parse them back into Spectrum or Chromatogram objects.
- File extension-based dispatch (.gz, .db, default .mzML) is the primary routing mechanism in `_open`, with optional content inspection for gzip type determination.
