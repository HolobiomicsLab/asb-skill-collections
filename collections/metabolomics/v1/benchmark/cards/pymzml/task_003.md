## Domain Knowledge
- Igzip is an indexed gzip format that embeds a seek table in the file header to enable rapid random access without decompressing the entire archive.
- The __getitem__ method in Python enables bracket-notation access (e.g., obj[key]) and is the standard interface for indexed retrieval in custom container classes.
- Binary file seeking requires precise byte-offset calculation; an embedded index maps logical item IDs to physical byte positions in the compressed stream.
- Sequential read functions must track state (current position) across calls to implement iteration semantics compatible with Python's iterator protocol.
