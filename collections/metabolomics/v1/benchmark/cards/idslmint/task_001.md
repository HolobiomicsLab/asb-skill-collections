## Domain Knowledge
- The transformer architecture consists of a stack of identical encoder layers, each containing a multi-head self-attention mechanism and a position-wise feed-forward network with residual connections and layer normalization.
- Positional encoding must be added to input embeddings to inject sequence order information, as transformer models lack inherent sequential processing capability.
- Mass spectrometry input (m/z and intensity pairs) must be converted into fixed-dimension tensor embeddings compatible with transformer input dimensionality (typically 512 or 768 features).
- Multi-head attention splits the embedding space into multiple subspaces to capture diverse spectral feature correlations in parallel, with typical configurations using 8–16 heads.
- The model forward pass must be run in evaluation mode (no gradients, dropout disabled) to verify architectural correctness and output stability.
