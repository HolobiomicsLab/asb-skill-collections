## Domain Knowledge
- MIST-CF employs an energy-based formula transformer architecture that ranks multiple candidate formulas and adduct types without computing fragmentation trees, relying instead on learned data-dependent representations.

- Top-k accuracy (particularly top-1 and top-3) is the standard ranking metric for formula assignment benchmarks, measuring whether the ground-truth formula appears in the top-k ranked predictions.
- MIST-CF operates in a de novo setting for precursor mass annotation from MS/MS data, meaning no reference spectrum database is consulted during ranking.
- Performance comparison against SIRIUS requires running both tools on identical benchmark spectra and directly comparing their ranking accuracy metrics to validate MIST-CF's improvements.
