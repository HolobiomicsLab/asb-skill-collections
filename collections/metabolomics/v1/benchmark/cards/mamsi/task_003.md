## Domain Knowledge
- Edge weights in metabolomics structural networks are assigned numerically to represent biological link types: isotopologue patterns (weight 1) indicate same molecular structure at different mass isotopes; adduct variants (weight 5) represent the same neutral mass in different ionization states; cross-assay links (weight 10) connect features across different LC-MS assays using [M+H]+/[M-H]- mass-neutral equivalence; correlation cluster co-membership (weight 15) groups features by statistical covariance.
- NetworkX graph serialization for Cytoscape import requires explicit node and edge attribute dictionaries (feature ID, m/z, RT, cluster assignments, compound annotations) to preserve metadata during round-trip curation and reimport.
- Node coloring by correlation cluster in pyvis visualization distinguishes statistical groupings (determined by hierarchical clustering on Pearson/Kendall/Spearman correlation matrices) from structural groupings (isotopologue + adduct), enabling visual separation of mechanistic vs. statistical relationships.
- The include_all parameter controls graph sparsity: when False, only features with at least one structural or correlation edge are rendered, reducing visual clutter; when True, isolated features appear as singleton nodes, useful for detecting unannotated singletons.
- Interactive HTML output from pyvis is browser-based and allows runtime manipulation (panning, zooming, node dragging); NetworkX object export supports offline analysis, batch curation scripts, and integration with external graph tools.
