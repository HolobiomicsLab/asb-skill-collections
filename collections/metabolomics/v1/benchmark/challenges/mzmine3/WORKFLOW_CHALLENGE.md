## Background

mzmine provides a user-friendly and extensible platform for mass spectrometry data analysis, architected with a complete set of modules spanning the entire MS data analysis workflow. The software is designed to support liquid chromatography (LC), gas chromatography (GC), ion mobility spectrometry (IMS), MS imaging (such as MALDI), and most MS instruments. Development builds are configured and accessible through the mzmine/mzmine repository's GitHub Actions, with the dev_build_release.yml workflow operational and linked via status badge.

## Research questions

- What is the complete set of processing modules provided by mzmine, and does each supported separation/ionisation technique (LC, GC, IMS, MS imaging) have at least one corresponding module in the software architecture?
- Does the dev_build_release.yml GitHub Actions workflow for the mzmine/mzmine repository complete successfully and produce build artifacts?

## Methods overview

Clone or access the mzmine GitHub repository to retrieve the complete source code and documentation structure.
