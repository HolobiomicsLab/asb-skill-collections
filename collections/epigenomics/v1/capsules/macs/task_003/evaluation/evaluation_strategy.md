# Evaluation Strategy

## Direct Checks

- verify file CTCF_PE_ChIP_chr22_50k.bedpe.gz exists in github:macs3-project__MACS test directory
- script_runs: macs3 callpeak executed on CTCF_PE_ChIP_chr22_50k.bedpe.gz in paired-end mode (--format BEDPE flag or equivalent)
- file_exists: narrowPeak output artifact generated by callpeak
- file_format_is: output narrowPeak file conforms to UCSC narrowPeak specification (chrom, chromStart, chromEnd, name, score, strand, signalValue, pValue, qValue, peak)
- value_in_range: reported average fragment length metric is approximately 253 bp (robust to ±5% tolerance)
- output_matches_reference: narrowPeak coordinates and peak counts align with expected output from paired-end CTCF analysis, no canonical answer—multiple valid peak calls depending on statistical threshold

## Expert Review

- verify that fragment length estimation (d) for paired-end CTCF data is biologically plausible and consistent with ChIP-Seq expectations for the CTCF protein
- assess whether peak quality (signal-to-noise, enrichment fold-change) is consistent with known CTCF binding patterns on chromosome 22
