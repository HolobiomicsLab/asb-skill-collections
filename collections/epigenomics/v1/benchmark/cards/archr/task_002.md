## Research Question
How does ArchR conditionally invoke monocle3 versus Slingshot trajectory analysis functions based on user selection?

## Connected Finding
ArchR implements trajectory analysis through three functions: getMonocleTrajectories and addMonocleTrajectory for Monocle3-based analysis, and addSlingShotTrajectories for Slingshot-based analysis, allowing users to select their preferred trajectory method.

## Task Description
Implement conditional trajectory analysis on a prepared ArchR project by invoking either addMonocleTrajectory (Monocle3) or addSlingShotTrajectories (Slingshot) based on user selection, producing a trajectory embedding object.

## Inputs

- User-specified trajectory method selection (monocle3 or Slingshot)

## Expected Outputs
