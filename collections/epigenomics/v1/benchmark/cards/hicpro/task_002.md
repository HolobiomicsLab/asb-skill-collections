## Domain Knowledge
- The iced module performs Iterative Correction and Eigenvector decomposition (ICE), a fundamental normalization technique for Hi-C contact matrices to correct for systematic biases.
- As of HiC-Pro version 3.X, iced is no longer bundled with HiC-Pro source code and must be installed independently before Hi-C processing workflows can execute normalization steps.
- iced depends on numerical Python libraries (numpy ≥1.18.1, scipy ≥1.4.1) which must be pre-installed or installed as transitive dependencies during iced setup.
- Hi-C contact matrices require normalization to account for sequencing depth, GC content biases, and other technical artifacts that distort downstream contact frequency estimates.
