## Domain Knowledge
- Juicer is a platform designed for analyzing Hi-C data at kilobase resolution, which measures 3D chromatin interactions across the genome.
- The feature annotation step is distinct from upstream map generation; it takes an already-processed .hic file as input and applies computational algorithms to detect structural features.
- Juicer command-line tools can produce multiple annotation types including loop calls (point-to-point interactions) and domain calls (contiguous chromatin regions with elevated internal contact frequency).
- The .hic binary format is Juicer's standard container for normalized, multi-resolution contact matrices; it enables rapid feature querying at different resolutions.
