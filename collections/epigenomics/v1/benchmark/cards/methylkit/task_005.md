## Domain Knowledge
- Gene annotation BED files contain intervals for promoters, exons, and introns; annotateWithGeneParts() overlaps differentially methylated bases with these features to classify their genomic context.
- CpG islands (typically >300 bp, >50% GC, >0.6 CpG/GpC ratio) and their flanking shores (up to 2 kb) are tracked as distinct classes because methylation patterns differ between island cores and shores.
- Percentage overlap is computed as the count of differentially methylated bases falling within each feature class divided by total bases, expressed as a percentage.
- The hg18 (human genome build 18) coordinate system used in the bundled annotation files must match the reference genome version used in the upstream bisulfite sequencing alignment and calculateDiffMeth() call.
