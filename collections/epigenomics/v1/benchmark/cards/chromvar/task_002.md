## Research Question
How can motifs be ranked by their variability in chromatin accessibility across cell populations, and which motifs show statistically significant differential deviation between distinct cell types?

## Connected Finding
chromVAR is designed to identify motifs or other genomic annotations associated with variability in chromatin accessibility between individual cells or samples, enabling ranking and differential analysis of motif usage.

## Task Description
Compute variability rankings and differential-deviation statistics for motifs using a chromVARDeviations object derived from the example 10-GM / 10-H1 dataset. Output a ranked variability table and a differential-deviation results table with per-motif statistics comparing cell types.

## Domain Knowledge
- Variability in chromVAR is measured as the standard deviation of bias-corrected z-scores across all cells/samples for a given motif, reflecting how consistently that motif's chromatin accessibility deviates from expectation.
- Bootstrap resampling in chromVAR variability testing redraws cells/samples with replacement to generate confidence intervals and assess whether observed variability exceeds the null hypothesis of variability = 1 (no effect).
- The differentialDeviations function tests whether bias-corrected deviations for a motif differ significantly between groups (e.g., GM vs H1 cell types) specified by colData column annotation, identifying cell-type-specific chromatin accessibility patterns.
- Bias-corrected deviations normalize for GC content and average peak accessibility, ensuring that variability reflects true biological chromatin-accessibility differences rather than technical biases.
