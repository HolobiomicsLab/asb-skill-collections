## Domain Knowledge
- tximport supports multiple upstream quantification methods; the type argument specifies which software generated the input files (e.g., 'oarfish', 'salmon', 'kallisto').
- The txOut=TRUE parameter retains transcript-level output; setting txOut=FALSE (default) aggregates to gene level via a tx2gene mapping.
- oarfish is a rust-based quantification tool that outputs quant.gz files containing per-transcript abundance and count estimates.
- The returned list contains three matrices: 'abundance' (TPM or similar normalized units), 'counts' (estimated fragment counts), and 'length' (transcript lengths in base pairs or median nucleotide count).
- Transcript-level length matrices account for potential differential isoform usage across samples, which is essential for accurate gene-level offset calculation in downstream differential analysis.
