## Domain Knowledge
- PAGA requires a prior neighborhood graph (computed via pp.neighbors) and cluster assignments (from tools like Leiden or Louvain); it then abstracts these into a coarse-grained graph where each node represents a cluster.
- The PAGA connectivity matrix stored in adata.uns['paga']['connectivities'] has shape (n_clusters, n_clusters) and represents edge weights between cluster-level nodes in the abstracted graph.
- Leiden clustering partitions cells into discrete groups; this partition is required as input to PAGA via the groupby parameter (typically adata.obs['leiden'] after running sc.tl.leiden).
- The paul15 dataset is a hematopoietic progenitor cell differentiation study commonly used as a benchmark for trajectory inference and clustering methods in Scanpy.
- PAGA is sensitive to the resolution and stability of the upstream clustering; different Leiden resolutions will produce different partition sizes and thus different PAGA graph structures.
