## Domain Knowledge
- DESeq2 expects un-normalized integer counts (not normalized values) as input to ensure correct assessment of measurement precision via the negative binomial model.
- The design formula must be specified before running DESeq(); for the airway dataset the formula ~cell+dex indicates cell type is a blocking factor and dex (dexamethasone treatment) is the variable of interest.
- By default, results() performs automatic independent filtering on mean normalized counts to optimize power for FDR control, and the alpha parameter (default 0.1) sets the target FDR level for this filtering.
- Factor levels are ordered alphabetically by default in R; the reference level for contrasts must be explicitly set or the results() contrast argument used to specify the numerator and denominator groups.
- Adjusted p-values (padj) account for multiple testing via the Benjamini-Hochberg method and are the appropriate threshold for significance claims in differential expression studies.
