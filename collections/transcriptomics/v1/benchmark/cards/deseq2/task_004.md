## Domain Knowledge
- Independent filtering in DESeq2 automatically optimizes the number of genes achieving a target FDR threshold by filtering based on the mean of normalized counts, not on p-values themselves.
- IHW (Independent Hypothesis Weighting) weights hypotheses adaptively using an auxiliary covariate (typically mean count) to improve power while controlling FDR under positive regression dependency assumption.
- The results() function with default alpha=0.1 sets genes with low mean count to NA in the padj column, reflecting filtering; IHW instead outputs weights and an FDR-controlled rejection set that may differ from the default independent filtering behavior.
- DESeq2's negative binomial GLM model corrects for library size internally via size factors, so raw counts (not normalized) must be provided as input to preserve measurement precision.
- When comparing filtering strategies, genes with all zero counts across samples will have baseMean=0 and all statistics set to NA; these are filtered regardless of method.
