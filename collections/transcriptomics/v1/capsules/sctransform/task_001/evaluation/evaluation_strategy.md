# Evaluation Strategy

## Direct Checks

- Confirm that every cited input artifact exists or is explicitly marked missing.
- Confirm that expected outputs are tied to figures, tables, files, or claims in the package.

## Llm Or Expert Review

- Judge whether the task description matches the source evidence.
- Judge whether the workflow description is sufficient for a future agent attempt.
- Review uncertainty notes and missing-information claims.

## Not In Scope For Mvp

- Running scientific code or reproducing figures.
- Scoring model-generated scientific outputs.
